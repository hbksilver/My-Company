Implementation/Analysis of an Intrusion Detection/Prevention System:

You will need to perform the following tasks in order to write your report (note: be sure to take screenshots of your steps along the way): Research the three (3) modes Snort is capable of running. Run Snort in each of these modes (one at a time with the default rules) and generate some traffic while it is running each time (e.g. web browser traffic, video game traffic, or anything else). Note the differences in the three modes and the types of traffic you noticed while running each mode. To begin with mode 1; Sniffer Mode, is a mode that reads network packets and output it to the user in live stream on the screen. In order to run sniffer mode first we run the command snort -v -i# to show TCP/IP packet headers in the console. Where number is the network adapter that appears after using the command snort -W. this command runs snort as packet sniffer. The type of packets in the mode is TCP/IP packet. There is also UDP packets, details are in the following screen shot named output 1:

The second Mode is packet logger mode. This mode is used when we are trying to record all packets to the disk. In this case we need to a locate logging directory, that’s when snort will switch to the mode. To do this we create a directory named lo using mkdir command, then, run the following command: Snort -dev -1 log -h my home IP address. Replace my home IP address with the real IP address of the network. The result is in output 4. Or run the command C:\>snort -vde -l c:\Snort\log -i2 . In this mode snort gatherall packets that have been discovred and put them in in it’s log directory. The files that being stored to are ASCII file. Third, and lastly, snort as Network Intrusion Detection Mode NIDS. In this mode snort provides sets of rules and filters that help find suspicious traffic. Then later use this information in cyber security analysis. Snort will rewrite these data and send them to script code to examine if they have malicious purpose. These data include packet streams of the sniffed traffic that have been stored using packet logger mode. According to Gigatux: “while Snort is a passable network sniffer, it is an excellent tool to detect intruders. When used as a NIDS, Snort provides newer real-time intrusion detection capabilities” (Snort as an NIDS: Quick and Dirty). From all three modes mention we can see the connections between them. We need to sniff packets in order to see what protocol are being used in the targeted traffic. This been done using the packet sniffing mode. Then, we use packet logger to record packets. The NIDS mode is used to research on that data collected by the previous tow modes to do forensic analysis. You will want to ensure the protocol/port you are detecting is open and/or listening on your Snort machine. Verify your rule works by running Snort in NIDS mode and launching a connection/scan/attack from machine via that protocol/port number. The alert generated upon detection should be easily identifiable on the screen or in a log file. Tip: You may find Snort to be easier to install and configure on Linux vs Windows, although both options are acceptable. Have I ever been hacked?

If you receive complaints from your co-workers, family members, friends, etc. about receiving spam E-mails from you, that may indicate that you have been compromised. Pop up widows from unknown programs asking permission to access certain tools or programs. If your anti-virus program or your firewalls have ben uninstalled or disabled, then your system have been hacked. (1) Solve the problem? Use antivirus system. If your task manager, or registry editor is disabled and can’t restart it. (2) Stop visiting untrusted webpages to solve the issue. USA Today (2018) list of signs your computer or phone has been hacked are: So many program or app crashes. If you are using internet and your data usage is more than usual. What is the Dark Web

The Dark web consists of multiple websites that use anonymity tools such as Tor browser and VPNS that enable users to surf the web privately. Most of the actions done in dark web considered illegal, such as child pornography, illegal drug sales, and hacking tools. The site attracts other groups of politicians and organizations. According to the Tech Republic: “Hacking is a fact of life for business and consumers alike. Often, leaked data surfaces and is sold to miscreants—hackers, shady government organizations, and other bad actors—on the Dark Web” (Tech Republic) We can conclude that the dark web has more than just buying weapons and illegal drugs. The website also states that: “The Dark Web—or darknet, backweb, onionweb—is frequently misunderstood. The network is used by legitimate actors like law enforcement organizations, cryptologists, and journalists as often as by malefactors and criminals” (Tech Republic). Criminals and professionals use of dark web is a like.
